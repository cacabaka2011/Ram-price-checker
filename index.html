Bourse RAM DDR5

Suivi en temps réel des prix (32GB 6000MHz CL30)

Pourquoi ce site ?

Les prix des composants informatiques varient chaque jour. Ce tracker a été créé pour identifier le meilleur moment pour acheter un kit de RAM DDR5 sans payer le prix fort.

Comment ça marche ?

Un robot (Python + Playwright) se connecte automatiquement toutes les 15 minutes sur Bing Shopping. Il analyse les résultats, calcule une moyenne des prix les plus pertinents et met à jour ce graphique.

Techno utilisées

Hébergé sur GitHub Pages. L'automatisation tourne via GitHub Actions. Le graphique est généré par la librairie ApexCharts.
